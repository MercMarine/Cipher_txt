#  Cipher text

Современное кроссплатформенное десктопное приложение для безопасного шифрования и дешифрования файлов. Использует военный стандарт шифрования **AES-256-GCM**, гарантируя полную конфиденциальность и целостность ваших данных.

> **Принцип приватности:** Все процессы шифрования и дешифрования происходят **локально** на вашем устройстве. Ваши файлы и пароли никогда не передаются в интернет.

---

## Возможности

- **Надежное шифрование:** Алгоритм AES-256-GCM с деривацией ключа через PBKDF2 (600,000 итераций).
- **Работа с любыми файлами:** Поддержка `.txt`, `.pdf`, `.jpg`, `.zip` и любых других форматов.
- **Встроенная читалка:** Мгновенный просмотр расшифрованного текста прямо в интерфейсе приложения.

---

##  Установка и запуск

### Вариант 1: Готовые исполняемые файлы (Рекомендуется)
Скачайте последнюю версию для вашей операционной системы со страницы [Releases](https://github.com/MercMarine/Cipher_txt/releases). Установка не требуется, просто запустите файл.

- **Windows:** `Cipher-text-v1.0.0-Windows.exe`

### Вариант 2: Запуск из исходного кода
Если вы хотите запустить приложение из исходников или внести изменения:

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/MercMarine/Cipher_txt.git
   cd Cipher_txt
   ```
2. Создайте и активируйте виртуальное окружение:
    ```
    python -m venv .venv
    # Для Windows:
    .venv\Scripts\activate
    # Для macOS/Linux:
    source .venv/bin/activate
    ```
3. Установите зависимости:
    ```
    pip install -r requirements.txt
    ```
4. Запустите приложение:
    ```
    python app.py
    ```

## Как пользоваться

1. Нажмите кнопку "Select a file" и укажите файл, который хотите зашифровать (или уже зашифрованный .enc файл для расшифровки).
2. Введите надежный секретный пароль.
3. Нажмите "Encpypt the file" или "Decrypt the file".
4. Выберите место для сохранения результата.
5. (Опционально) Если вы расшифровали текстовый файл, его содержимое автоматически отобразится во встроенной читалке внизу окна.

## Сборка исполняемого файла (Для разработчиков)

Проект использует PyInstaller для упаковки в один файл. Выполните в терминале:
```
pip install pyinstaller
```
# Для Windows:
```
pyinstaller --noconfirm --onefile --windowed --icon=icon.ico --name "Cipher text" app.py
```
