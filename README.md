#  Cipher text

Современное кроссплатформенное десктопное приложение для безопасного шифрования и дешифрования файлов с поддержкой нескольких криптографических алгоритмов.

> **Принцип приватности:** Все процессы шифрования и дешифрования происходят **локально** на вашем устройстве. Ваши файлы и пароли никогда не передаются в интернет.

---

## Возможности

### Надежное шифрование

- **AES-256-GCM** - военный стандарт шифрования
- **ChaCha20-Poly1305** - современный высокоскоростной шифр
- Гибкий выбор алгоритма под ваши задачи

### Деривация ключей
- **PBKDF2** (600,000 итераций) - проверенный стандарт
- **Argon2id** - победитель конкурса Password Hashing Competition, устойчив к GPU-атакам

### Работа с файлами
- Поддержка **любых форматов**: `.txt`, `.pdf`, `.jpg`, `.zip`, `.docx` и других
- Автоматическое определение алгоритмов шифрования при расшифровке
- Сохранение целостности данных

##  Установка и запуск

### Вариант 1: Готовые исполняемые файлы (Рекомендуется)
Скачайте последнюю версию для вашей операционной системы со страницы [Releases](https://github.com/MercMarine/Cipher_txt/releases). Установка не требуется, просто запустите файл.

- **Windows:** `Cipher-text-v1.1.0-Windows.exe`

### Вариант 2: Запуск из исходного кода
Если вы хотите запустить приложение из исходников или внести изменения:

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/MercMarine/Cipher_txt.git
   cd Cipher_txt
   ```
2. Создайте и активируйте виртуальное окружение:
    ```
    python -m venv .venv
    # Для Windows:
    .venv\Scripts\activate
    # Для macOS/Linux:
    source .venv/bin/activate
    ```
3. Установите зависимости:
    ```
    pip install -r requirements.txt
    ```
4. Запустите приложение:
    ```
    python app.py
    ```

## Как пользоваться

1. Выберите файл: Нажмите кнопку "Select a file" и укажите файл, который хотите зашифровать (или уже зашифрованный .enc файл для расшифровки).
2. Выберите алгоритмы:
    - Ciphering algorithm: AES-256-GCM или ChaCha20-Poly1305
    - KDF (Key Derivation Function): PBKDF2 или Argon2id
3. Введите пароль: Введите надёжный секретный пароль.
4. Зашифруйте/Расшифруйте: Нажмите "Encrypt the file" или "Decrypt the file".
5. Сохраните результат: Выберите место для сохранения зашифрованного/расшифрованного файла.
6. Просмотр текста: Если вы расшифровали текстовый файл, его содержимое автоматически отобразится во встроенной читалке внизу окна.

## Сравнение алгоритмов
### Шифрование:

- AES-256-GCM - аппаратное ускорение на современных процессорах, стандарт для государственных организаций
- ChaCha20-Poly1305 - высокая скорость на устройствах без AES-NI, используется в TLS 1.3

### Деривация ключей:
- PBKDF2 - широко поддерживается, проверен временем
- Argon2id - лучшая защита от брутфорс-атак, рекомендуется для максимальной безопасности

## sФормат зашифрованного файла
Файлы сохраняются с расширением .enc и имеют следующую структуру:
- Байт 0: Версия формата (0x01)
- Байт 1: Идентификатор KDF (0x01 - PBKDF2, 0x02 - Argon2id)
- Байт 2: Идентификатор шифра (0x01 - AES-GCM, 0x02 - ChaCha20)
- Байты 3-18: Salt (16 байт)
- Байты 19-30: Nonce (12 байт)
Остальные байты: Зашифрованные данные + тег аутентификации
